Eligibility Criteria at a Glance

  • PM-SC/ST: Marks ≥ 50% · Income ≤ ₹2.5L · SC/ST only
  • PM-OBC: Marks ≥ 55% · Income ≤ ₹1.5L · OBC only
  • National Merit: Marks ≥ 85% · Income ≤ ₹6L · All categories
  • PMSS: Marks ≥ 75% · Income ≤ ₹5L · All categories
  • CSS: Marks ≥ 80% · Income ≤ ₹4.5L · All categories
  • Top Class SC: Marks ≥ 70% · Income ≤ ₹6L · SC only
